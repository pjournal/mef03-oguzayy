BKM Assignment

This document is an analysis of BKM data for the periods of January 2019 to June 2019. Data can be reached from https://bkm.com.tr/

Importing the Data

The file is downloaded from the link and is assigned to a new variable called table.

url <- "https://bkm.com.tr/secilen-aya-ait-sektorel-gelisim/?filter_year=2019&filter_month=1"
page <- read_html(url)
table <- html_table(page, fill = TRUE)[[4]][-c(1:2),]


for(i in 2:6) {
  url <- paste("https://bkm.com.tr/secilen-aya-ait-sektorel-gelisim/?filter_year=2019&filter_month=", i, sep = "")
  page <- read_html(url)
  table <- bind_rows(table, html_table(page, fill = TRUE)[[4]][-c(1:2),-1])
}

temp1 <- c(table%>% select(X1) %>%  filter(X1 != "NA"))
temp2 <- c(rep(temp1[["X1"]], times=6))
temp3 <- as.character(c(rep(seq(1:26), times=6)))


tmpTable <- table %>% mutate(X1 = temp2) %>% filter(X1 != "TOPLAM") %>% mutate(X6=temp3)


month_1 <- c(rep(1:6, times=1, each=26))
table <- tmpTable %>% mutate(month = month_1)


table2  <- as.data.frame(lapply(table, function(x) as.numeric(gsub(",", ".", gsub("\\.", "", x)))))
table2[,1] <- tmpTable[,1]
table2[,6] <- tmpTable[,6]
table <- table2

names(table) <- c("Sector", "TransactionCount_CC", "TransactionCount_DC", "TransactionAmount_CC", "TransactionAmount_DC", "Sector_ID", "Month")

Analyses Related to the Data

  • The first thing is to find the sectors with the highest transaction amount for both debit and credit cards.
  • We can see that the sectors MARKET VE ALIVER MERKEZLER (Sector ID=17), GYM VE AKSESUAR (Sector ID=9), BENZN VE YAKIT STASYONLARI (Sector ID=3), ETL GIDA (Sector ID=5), HZMET SEKTRLER (Sector ID=11), YEMEK (Sector ID=25), ELEKTRK-ELEKTRONK EYA, BLGSAYAR (Sector ID=8) are the ones with the highest. So we can group the rest as OTHER.
  • We can now find the fraction of the sum of transaction amounts for each sectors.
  • Then we can visualise them in a pie chart.
  • We can conclude that the sector of MARKET VE ALIVER MERKEZLER has a grave share with respect to the rest of the sector.
table %>%
  filter (Month == 6) %>%
  transmute(Sector, Sector_ID, Tot_Tran_Amount = (TransactionAmount_CC + TransactionAmount_DC)) %>%
  arrange(desc(Tot_Tran_Amount)) %>%
  mutate(Sector_ID = case_when(Sector_ID %in% c("17","9","3","5","11","25","8") ~ Sector_ID, TRUE ~ "99")) %>%
  group_by(Sector_ID) %>%
  summarise(Tot_Tran_Amount = sum(Tot_Tran_Amount)) %>%
  mutate(Fra_Tran_Amount = round(Tot_Tran_Amount / sum(Tot_Tran_Amount)*100,2)) %>%
  ggplot(data = ., aes(x = "", y = Fra_Tran_Amount, fill = Sector_ID)) +
  geom_bar(width = 1, stat = "identity", color = "black") +
  coord_polar("y", start = 0)+
  geom_text(aes(x = "", y = Fra_Tran_Amount, label = Fra_Tran_Amount),position = position_stack(vjust = 0.5),color = "black")+
  labs(fill = "Sectors")+
  theme_void()

We can go deep with analyzing market sector with respoect to the months for 2019. There is an increase in the total transaction on May with respoect to other months.

table %>%
  filter (Sector_ID == 17) %>%
  mutate(Tot_Tran_Amount = (TransactionAmount_CC + TransactionAmount_DC)) %>%
  ggplot(data=., aes(x=Month, y=Tot_Tran_Amount)) +
  geom_bar(stat="identity", position=position_dodge(), fill="green")+
  labs(x="Month", y = "Total Transaction Amount for Markets") +
  theme_minimal()

As the graphs imply, all sectors have a peak at May.

table %>%
  group_by(Month) %>% summarize (Tot_Tran_Amount=sum(TransactionAmount_CC + TransactionAmount_DC)) %>%

  geom_bar(stat="identity", position=position_dodge(), fill="blue")+
  labs(x="Month", y = "Total Transaction Amount for all Sectors") +
  theme_minimal()
